## 📌 前提条件

-   1人日 = 1時間
-   想定チーム規模：2名（Backend中心）
-   見積単位：
    -   XS = 2h
    -   S = 4h
    -   M = 8h
    -   L = 16h
    -   XL = 24h以上
-   優先度：
    -   🔴 P0：OIDC成立に必須
    -   🟡 P1：UX向上
    -   🟢 P2：安定化
    -   ⚪ P3：拡張

# 🧮 計算式

総工数 ÷ (人数 × 40h/週)

例：

104h ÷ (2 × 40) = 約1.3週間

レビュー込みなら ×1.3推奨
