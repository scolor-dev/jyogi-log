# 技術スタック

---

## 概要
- 規模：小規模（~300 MAU）
- チーム：4人
- 可用性：99%
- 方針：シンプル構成・後から拡張

---

## フロントエンド

| 項目 | 技術 | 理由 |
|------|------|------|
| 言語 | TypeScript | 型安全 |
| FW | React | 実績・拡張性 |
| UI | Tailwind CSS | 開発速度 |
| 状態管理 | React Query | Server State管理 |
| フォーム | React Hook Form | 軽量・高速 |

---

## バックエンド

| 項目 | 技術 | 理由 |
|------|------|------|
| 言語 | Rust | 高性能・安全 |
| FW | axum | シンプル・型安全 |
| API | REST | 実装容易 |
| DBアクセス | sqlx | 型安全 |
| 認証 | JWT + OAuth2 Provider | 本システムの中核 |
| 非同期 | tokio | 標準 |

---

## データベース

| 項目 | 技術 | 理由 |
|------|------|------|
| DB | PostgreSQL | 安定・高機能 |
| キャッシュ | なし（必要時Redis） | 初期は不要 |
